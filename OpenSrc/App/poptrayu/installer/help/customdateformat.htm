Custom Date Format

The "Date" column of the main window can be customized to display the date/time with a custom-selected date format.

Slash (/) will be replaced by regional date seperator. If you wish to use a different seperator, enclose the deisred seperator in single or double quotes.

Additionally, there are shortcuts for specific date formats specified in the Windows Regional Settings:

Date Format Shortcuts
ddddd Short Date Format 7/10/2013
dddddd Long Date Format Wednesday, July 10, 2013

Showing Times:

 
Hour
Minute
Seconds
Milliseconds
No Leading Zero h  n  s  z 
Show Leading Zero hh  nn  ss  zzz 

Colon (:) will be replaced with the regional time seperator. If you wish to use a different seperator, enclose it in single or double quotes.

If you prefer 12 hour AM/PM time to 24 hour time, add one of these after your time format string:

Times also have shortcut formats for the regional long or short time formats:

Time Format Shortcuts
t Short Time Format 4:23 PM
tt Long Time Format 4:47:40 PM
c Short Date Format + Long Time Format 7/10/2013 4:23:58 PM
