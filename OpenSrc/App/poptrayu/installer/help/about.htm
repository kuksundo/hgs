About

PopTrayU has been developed by Jessica Brown. More information about PopTrayU can be found on it's website at http://sourceforge.net/projects/poptrayu/

PopTrayU is a derivative work of Renier Crause's PopTray, building upon the features available in PopTray 3.2. PopTray was proudly developed in South Africa. Answers to many questions on the PopTray Forum may be applicable to PopTrayU, unless it is in regards to a part of the application that has been changed. If you would like to report a bug in PopTrayU, please see the PopTrayU Website. Additionally, PopTray Plugins should be compatible with PopTrayU at this time.

Thank you Renier for creating PopTray!
