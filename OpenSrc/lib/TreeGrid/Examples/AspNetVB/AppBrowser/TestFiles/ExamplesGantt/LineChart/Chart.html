Line charts and XY points charts

X column contains informational row indexes that are not used in chart calculations, so any sorting change affects the left line chart. If the X column will be used as left line chart X sources, the sorting will not affect the chart.
Y1 and Y2 columns are editable columns used as Y sources for the left line chart. Their values can be also changed by mouse in the Gantt column: dragging changes the position, ctrl+click deletes the point, double click adds new point, if possible and right click shows popup menu.
Y1*Y2 and Y1-Y2 columns are calculated from the Y1, Y2 columns and are used also as next Y sources in the left line chart.
The right chart is XY points chart with two charts, the first has Y1*Y2 column as X sources and Y1-Y2 column as Y sources and the second chart has the sources swop.
